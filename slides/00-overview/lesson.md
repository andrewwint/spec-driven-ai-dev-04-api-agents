### What You're Really Learning

This course uses APIs and agents as the **vehicle**, but the real lessons are:

| Surface Topic | Actual Skill |
|---------------|--------------|
| Flask endpoints | Wrapping ANY function as a service |
| SQLite queries | Database patterns that transfer to any SQL |
| Strands agents | AI orchestration concepts (transfers to LangChain) |
| MCP servers | **Efficient documentation access** — the "10x Google Fu" skill |
| CDK basics | Infrastructure as code thinking |

**You are NOT becoming a Flask expert.** You're learning to:
- Use skills to access documentation efficiently
- Make informed API architecture decisions
- Build production-quality services with confidence
- Know when to hand off to platform specialists

## Fast Track (For API-Experienced Developers)

*Already comfortable with REST APIs, Flask/FastAPI, and databases?*

**Skip to the new concepts:**
1. **Module 1.3** — APIs as Context Boundaries (the throughline connection)
2. **Module 4** — Strands Agents (AI integration patterns)
3. **Module 5** — MCP Documentation Pattern ⭐ (the "10x Google Fu" skill — don't skip this)
4. **Module 6.2** — CDK vs Terraform decision framework

**Minimum path:** ~4 hours (Modules 1.3, 4, 5, 6.2, 7)

**What you'll still learn:**
- How to wrap ML models as API endpoints (Course 3 integration)
- Strands Agents for model-agnostic AI capabilities
- MCP servers for efficient documentation access (KEY differentiator)
- CDK introduction for infrastructure as code

> 💡 **Progressive Disclosure:** Even experienced developers should read `AGENTS.md` and `HISTORY.md` — they carry context from Courses 2-3 that agents reference.
